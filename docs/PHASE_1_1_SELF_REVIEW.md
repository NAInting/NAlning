## 三、治理自查

| 问题 | 判断 |
|---|---|
| 谁能看原始对话？ | 默认只有学生 Agent 运行时和授权审计路径；教师/家长不能看 |
| 谁能看学业摘要？ | 学生本人、授权任课教师可看必要摘要；家长只看周期性派生视图 |
| 谁能看情绪层？ | 不进入常规教师/家长视图，只允许 campus-local 判断与抽象信号 |
| 谁能改记忆？ | 学生可申请删除/修正普通记忆；治理规则保留最小必要审计 |
| 是否留痕？ | 高危升级、教师信号、关键访问应落审计，后续由阶段 1.2/1.6/7 落地 |
| 是否可能泄露隐私？ | 当前文档明确禁止原文、摘录、关键词、自由文本理由穿透到教师/家长 |

## 四、InterAgentSignal 自查

检查项：

1. 文档允许的教师信号为 `mastery_update`、参与度趋势、抽象求助、抽象情绪异常、干预建议代码。
2. 文档禁止教师看到原始对话全文、情绪表达原文、个人生活细节、未脱敏隐私层记录、Agent 内部推理链。
3. 情绪异常示例只包含 `severity`、`suggested_action`、`detected_at`。
4. 文档中的禁止示例包含 `conversation_excerpt` 和 `rationale_summary`，目的是说明不能这样设计，不是允许字段。

结论：通过。

## 五、记忆边界自查

| 记忆层 | 当前约束 |
|---|---|
| Working Memory | 只服务当前上下文，不做长期标签 |
| Episodic academic | 可生成教师必要摘要 |
| Episodic emotional | 只本地判断与抽象信号，不传原文 |
| Episodic personal | 默认学生私有，谨慎使用 |
| Semantic Memory | 按版本更新、学生可见、低置信度不得进入 |

结论：通过。
